# Design Guidelines - GuiselleArmstrong.com

## Brand Identity
- **Industry:** Governance, Risk, and Compliance (GRC)
- **Tone:** Professional, confident, approachable
- **Target Audience:** Recruiters, hiring managers, GRC professionals

## Color Palette Suggestions
- Primary: Deep blues or teals (trust, professionalism)
- Secondary: Grays and neutrals (clean, modern)
- Accent: Subtle greens (growth, progress)

## Layout Priority
1. Hero/Biography - Name, title, professional summary
2. Experience Timeline - Career progression with key metrics
3. Skills - Technical and soft skills
4. Certifications - Professional credentials
5. Education - Academic background
6. Contact/Resume Download

## Tips
- Keep navigation simple: Home, Experience, Projects, Contact
- Include resume download button prominently
- Ensure mobile-friendly layout
- High contrast for readability

## Verification Rule
- **Always verify visual changes using browser tools before pushing.** Launch the local Hugo server, use browser_action to scroll through each affected section, and confirm the layout matches the intended design on both desktop and mobile before creating a PR.
