# Design and Branding Guidelines - GuiselleArmstrong.com

## Brand Identity

### Professional Positioning
- **Industry:** Governance, Risk, and Compliance (GRC)
- **Specialization:** Process Improvement, Six Sigma, IT Compliance
- **Tone:** Professional, confident, approachable
- **Target Audience:** Recruiters, hiring managers, GRC professionals

### Color Palette
- **Primary:** Deep blues or teals (conveys trust, professionalism, stability)
- **Secondary:** Grays and neutrals (clean, modern look)
- **Accent:** Subtle greens (growth, progress - aligns with Six Sigma methodology)
- **Avoid:** Bright reds, overly casual colors

### Typography
- Use clean, professional fonts from the HugoBlox "academic" font pack
- Headings should be bold and clear
- Body text should be highly readable
- Keep font sizes consistent with theme defaults

## Layout and Structure

### Homepage Priority
1. **Hero/Biography Section** - Name, title, professional summary
2. **Experience Timeline** - Career progression with key metrics
3. **Skills Cloud** - Technical and soft skills
4. **Certifications** - Professional credentials
5. **Education** - Academic background
6. **Contact/Resume Download** - Easy ways to connect

### Navigation
- Keep menu simple: Home, Experience, Projects, Contact
- Include resume download button prominently
- Ensure mobile-friendly navigation

## Content Presentation

### Experience Section
- Use reverse chronological order (most recent first)
- Highlight measurable achievements with numbers
- Include company name, dates, location for each role
- Use bullet points for readability

### Skills Display
- Group by category: Core Competencies, Technical, Frameworks, Tools
- Show proficiency levels where applicable
- Keep descriptions concise

### Certifications
- List all certifications clearly
- Include issuing organization
- Show completion date if available

## Professional Imagery

### Profile Photo
- Use a professional headshot
- Ensure good lighting and neutral background
- Dress professionally (business attire)

### Icons and Graphics
- Use consistent icon style throughout
- Prefer simple, professional icons
- Avoid decorative or playful graphics

## Accessibility and Usability

### Readability
- Ensure high contrast between text and background
- Use sufficient font sizes
- Keep lines of text to a reasonable length

### Mobile Responsiveness
- Test all pages on mobile devices
- Ensure buttons and links are easily tappable
- Verify images scale appropriately

### Performance
- Optimize images for web
- Minimize use of heavy animations
- Ensure fast page load times

## Contact Information

### Display Rules
- Show email prominently
- Include LinkedIn profile link
- Include phone number
- Optionally show location (city, state)

### Call-to-Action
- "Download Resume" button on homepage
- "Contact Me" link in navigation
- Make it easy for recruiters to reach out

## Content Updates

### When Making Changes
1. Always backup current content before editing
2. Test changes locally before committing
3. Verify all links work after changes
4. Check mobile view after changes

### Regular Maintenance
- Update experience section with new roles
- Add new certifications as earned
- Keep skills section current
- Review and update professional summary periodically
